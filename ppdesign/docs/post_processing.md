# Post-Processing Utilities

## `ppdesign select`

Filters codon-aligned candidate probes generated by the gene-based pipeline. Core features:

- GC/Tm bounds with optional ViennaRNA-based structure screens.
- Degeneracy handling with IUPAC expansion limits.
- Ranking metrics emitted to `selected_probes.csv`.

Suggestions:

- Expand logging around discarded probes to make debugging easier.
- Add tests covering secondary structure filtering when ViennaRNA is not installed (currently silently skipped).

## `ppdesign rank`

Scores a pre-built panel against a reference genome to visualize coverage.

- Loads probe FASTA/CSV, maps them to reference using minimap2 or BLAST.
- Produces plots (PNG/HTML) in `results/<run>/plots/` and summary TSVs.

Suggestions:

- Provide an option to bypass expensive re-alignment when users already have mapping coordinates.
- Consolidate plotting code to handle large probe sets (>10k) with subsampling to avoid clutter.
